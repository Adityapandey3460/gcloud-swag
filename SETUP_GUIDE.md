## рџљЂ Step-by-Step Setup

### 1. Install Flutter
```bash
# Download Flutter SDK from flutter.dev
# Add to PATH, then:
flutter doctor   # Check everything is installed
```

### 2. Create the project
```bash
flutter create swag_admin_app
cd swag_admin_app
# Replace lib/ and pubspec.yaml with the provided files
```

### 3. Setup Firebase

#### A. Create Firebase Project
1. Go to [console.firebase.google.com](https://console.firebase.google.com)
2. Create project в†’ name: `gcloud-swag-2024`
3. Enable **Firestore** в†’ Start in production mode
4. Enable **Authentication** в†’ Email/Password
5. Add admin user: `admin@gcloud.com` with your password

#### B. Connect Firebase to Flutter
```bash
# Install FlutterFire CLI
dart pub global activate flutterfire_cli

# Login to Firebase
firebase login

# Configure - this generates firebase_options.dart automatically
flutterfire configure --project=gcloud-swag-2024
```
This creates `lib/firebase_options.dart` вЂ” **don't commit this to git**.

### 4. Firestore Rules
In Firebase Console в†’ Firestore в†’ Rules:
```
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /students/{studentId} {
      allow create: if true;        // Anyone can register
      allow read, update: if request.auth != null;  // Only admin can verify/claim
    }
  }
}
```

### 5. Install dependencies & run
```bash
flutter pub get
flutter run   # Connect Android phone via USB (enable USB debugging)
```

### QR Code Content
The QR code encodes the **Firestore document ID** of the student.
When admin scans в†’ app calls `FirebaseService.getStudentById(scannedId)`.

The student registration website should generate QR codes like:
```dart
// QR data = Firestore document ID
QRCode.toDataURL(docId)  // On web
```
